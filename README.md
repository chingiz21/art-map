# art-map

## Description

Web-application, which allows you to explore the world and open new places!

## Used technologies:

- NodeJS
- ExpressJS
- Axios
- React
- TypeScript
- React-router
- Mobx
- Sequelize
- PostgreSQL
- Leaflet

## Folders

- <code>/backend</code> - server side (MVP)
- <code>/client</code> - client side
